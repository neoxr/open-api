## OPEN-API

> Lightweight rest api web base script with various superior features :v

### Features

- [x] Plugin as Router
- [x] Auto Load Scraper
- [x] Auto List Feature
- [x] Pagination
- [x] High Optimation
- [x] Using Jomok Engine
- [x] Easy to Customize
- [x] etc.

### Installation & Run

```
$ yarn
$ node .
```

or want to use pm2

```
$ yarn
$ npm i -g pm2
$ pm2 start index.js && pm2 save && pm2 logs
```
