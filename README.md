## OPEN-API

> A TypeScript-based lightweight website script using a plugin-powered router that's easy to customize and optimized for REST APIs.

### ✨ Features

- [x] **Plugin-Based Router** – Modular and flexible routing system
- [x] **High Performance** – Lightweight and fast response
- [x] **Easy Customization** – Clean architecture for rapid changes
- [x] **User IP Display** – Shows visitor IP via built-in endpoint
- [x] **REST API Ready** – Suitable for microservices or backend APIs
- [x] **Simple Setup** – Minimal dependencies, quick to deploy
- [x] **Fully Typed** – Built with TypeScript for type safety
- [x] ...and more!
